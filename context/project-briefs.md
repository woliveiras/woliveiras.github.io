# Project Briefs

## Project 1: Veterinary History Agent

Objective:

> Create an agent that organizes exams, vaccines, medications, clinical events, and tutor notes into a timeline that supports veterinary consultation.

Capabilities:

- Pet registration.
- Document ingestion.
- Date extraction.
- Exam extraction.
- Medication list.
- Vaccine history.
- Weight history.
- Temporal comparison.
- Clinical event summary.
- Questions for the veterinarian.
- Citations.

Expected output:

```text
Veterinary summary
Timeline
Current medications
Relevant exams
Vaccines
Weight evolution
Points of attention
Questions for the veterinarian
Warning signs
```

## Project 2: Veterinary Pre-Consultation Agent

Objective:

> Create an agent that transforms scattered information into a structured briefing for the veterinary team before the consultation.

Input:

- Main consultation reason.
- Signs observed by the tutor.
- Documents.
- History.
- Medications.
- Recent events.
- Tutor questions.

Expected output:

```text
Veterinary consultation briefing
1. Main reason
2. Timeline
3. Relevant history
4. Documents used
5. Suggested questions
6. Points to confirm
7. Warning signs
```

This is the main project of the series.

## Project 3: Internal Agent for Veterinary Clinics

Objective:

> Create an operational support agent for reception, veterinarians, and technical staff to inspect history, documents, and pending items with permission controls.

Capabilities:

- Search history by animal.
- Summarize previous consultations.
- Locate documents.
- Identify consultation pending items.
- Identify overdue or upcoming vaccines.
- Enforce role-based access.
- Produce audit trail.

Expected output:

```text
Internal panel
1. Recent history
2. Relevant documents
3. Vaccines and medications
4. Administrative pending items
5. Safety alerts
6. Actions allowed for this role
```

## Project 4: Veterinary Education Agent

Objective:

> Create an agent that answers general questions with trusted veterinary sources without personalizing diagnosis or replacing professional guidance.

Example questions:

- "What can cause vomiting in cats?"
- "What is bruxism in dogs?"
- "How should I prepare for a consultation?"
- "What should I observe when my cat stops eating?"
- "What general care is recommended after vaccination?"

Characteristics:

- Uses curated sources.
- Cites references.
- Avoids diagnosis.
- Separates general information from personal advice.

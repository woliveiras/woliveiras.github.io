# Writing Roadmap

## Phase 1: Define Scope

Deliverables:

- Series thesis.
- Target audience.
- Outline.
- Main practical project.
- Technical stack.
- Editorial tone.
- Safety limits.

Expected result:

> A 3 to 5 page editorial proposal.

## Phase 2: Build the Base Project

Deliverables:

- Repository.
- Database schema.
- Seeds for veterinary scenarios.
- Harness commands for reset and data loading.
- Document ingestion commands.
- Simple parser.
- pgvector indexing.
- Question and answer command.
- Veterinary pre-consultation command.
- Basic citations.

Expected result:

> A minimal prototype, executed through a local agent harness, that accompanies the first posts.

## Phase 3: Write Part 1 and Part 2

Deliverables:

- Foundation posts.
- Ingestion posts.
- Veterinary examples.
- Diagrams.
- Checklists.

Expected result:

> The conceptual and structural base of the series.

## Phase 4: Evolve Retrieval and Agents

Deliverables:

- Hybrid search.
- Reranking.
- SQL retrieval.
- Timeline.
- Router.
- LangGraph agent.
- Safety check.

Expected result:

> The system moves from simple RAG to Agentic RAG.

## Phase 5: Add Production Concerns

Deliverables:

- Evaluation.
- Test dataset.
- Structured logs.
- Tracing.
- Prompt injection tests.
- Privacy checklist.
- Cost and performance analysis.

Expected result:

> The series moves from tutorial to applied engineering.

## Phase 6: Finish Practical Projects

Deliverables:

- Veterinary history agent.
- Veterinary pre-consultation agent.
- Internal clinic agent.
- Veterinary education agent.
- Product roadmap.

Expected result:

> A complete series with reusable projects.
